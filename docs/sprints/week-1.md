# Спринт 1: Базовый UI Канбан-доски

## Цель: 
Сделать красивый и современный UI для управления карточками без реального бекенда.

## Задачи:
- [ ] **Task 1**: Настроить CSS-переменные (цвета, шрифты) и глобальные стили (`index.css`). Внедрить темную тему с премиальным дизайном (Glassmorphism, glow эффекты).
- [ ] **Task 2**: Создать компонент `Board`, который рендерит колонки: Backlog, To Do, Agent Thinking, Human Review, Done.
- [ ] **Task 3**: Создать компонент `Card` с отображением: Названия задачи, Описания (промпта), Аватара назначенного агента.
- [ ] **Task 4**: Сделать заглушки для 3-х типов агентов (Architect, Coder, Reviewer).
