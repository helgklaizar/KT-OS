# Описание проекта
Agent-Kanban — это панель управления для мульти-агентной системы, представленная в виде канбан-доски. 

# Текущее состояние
- [x] Инициализация репозитория.
- [x] Скаффолдинг frontend (Vite + React + TS).
- [ ] Базовый UI доски (Колонки, Карточки).

# Важные решения
- Принята архитектура Web App + Local API. Фронтенд на Vite (React), бекенд и ИИ-агенты будут на FastAPI (или Node.js).
