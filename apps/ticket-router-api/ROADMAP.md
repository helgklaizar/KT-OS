# ROADMAP

## Общая цель
Пройти путь от основ Python до создания полноценной Production-ready ML-системы.

### Phase 1: Data Foundations (Недели 1-2)
- [ ] Разворачивание среды и знакомство с инфраструктурой Python (pip, venv/conda, jupyter).
- [ ] Продвинутый Python (типизация, ООП, генераторы).
- [ ] Сбор данных: веб-скрейпинг (BeautifulSoup) и работа с API.
- [ ] Предобработка и анализ (EDA): NumPy, Pandas. Сохранение в SQL/PostgreSQL.
- [ ] Визуализация: Matplotlib, Seaborn, интерактивный Plotly.

### Phase 2: Classic Machine Learning (Недели 3-4)
- [ ] Классические алгоритмы: линейная/логистическая регрессия, KNN, SVM (scikit-learn).
- [ ] Деревья решений и Ансамбли: Random Forest, XGBoost, LightGBM, CatBoost.
- [ ] Кросс-валидация, подбор гиперпараметров (Optuna).
- [ ] Метрики качества (ROC-AUC, F1, RMSE).

### Phase 3: Deep Learning (Недели 5-6)
- [ ] Введение в PyTorch: тензоры, градиенты, архитектура нейросети.
- [ ] CV (Computer Vision): Сверточные сети (CNN), ResNet, Transfer Learning.
- [ ] NLP: Рекуррентные сети (LSTM), механизм Attention.
- [ ] Hugging Face Transformers (BERT и аналоги).

### Phase 4: MLOps & Production (Недели 7-8)
- [ ] Итоговая ML-система (выбранный проект из implementation_plan).
- [ ] Оборачивание модели в REST API (FastAPI).
- [ ] Docker-контейнеризация сервиса.
- [ ] Создание интерфейса (Streamlit).
